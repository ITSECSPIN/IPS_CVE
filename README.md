# Internal Security Signatures

**⚠️ SECURITY WARNING: STRICTLY CONFIDENTIAL ⚠️**

This repository contains sensitive vulnerability signature configurations and blocklists used by the internal Security Scanner.

## Access Control & Compliance
*   **Classification:** **INTERNAL USE ONLY**
*   **Visibility:** This repository must remain **PRIVATE** at all times.
*   **Sharing:** Do not fork, export, or share these contents with third parties or public AI models.
*   **Access:** Automated access is granted strictly via **Read-Only Personal Access Tokens (PAT)** or scoped Deploy Keys.

## Risk Context
While this repository may only contain CVE IDs (e.g., `CVE-202X-XXXX`), this list constitutes a **Defensive Map** of our environment. Exposing it allows potential attackers to analyze our coverage gaps and tailor exploits to bypass our specific IPS signatures.

**Treat this data with the same care as credentials or API keys.**
